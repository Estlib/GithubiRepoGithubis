Trello - Juhan

Ajaloost

Trello on projektihalduse tööriist, mille arendamist alustas 2010. aastal ettevõte Fog Creek Software. Selle lõi Michael Pryor ja Joel Spolsky eesmärgiga pakkuda lihtsat ja visuaalset viisi projektide haldamiseks. Avalikult lansseeriti Trello 2011. aastal TechCrunch Disrupt konverentsil. 2017. aastal ostis Trello ettevõtte Atlassian ning arendus jätkub aktiivselt tänapäevani.

Peamised funktsioonid

Tahvlid (Boards)

Trello põhineb tahvlitel, mis aitavad organiseerida projekte ja töövooge visuaalsel kujul.

Kaardid (Cards)

Kaardid esindavad ülesandeid. Neile saab lisada kirjeldusi, tähtaegu, faile ja kommentaare.

Koostöö

Kasutajad saavad kutsuda meeskonnaliikmeid, määrata ülesandeid ja jälgida edenemist reaalajas.

Visual Studio Code

Visual Stuido Code on mugav koodiredaktor, mis toetab palju erinevaid keeli. VSCode võimaldab
luua projekti arhitektuuri, lähtekoodi kirjutada ja hallata, debagida ja kasutada laiendusi tasuta.
VSCode arendus algas aastal 2014 ja esimene stabiilne versioon ilmus 2016 aasta aprillis. VSCode loodi, kuna
Microsoft soovis luua kerge, kiire ja mitmel operatsioonisüsteemil töötava tööriista. VSCode-i arendas
peamiselt Microsofti arendustiim.

RustRover - Enrique

Tutvustus


RustRover on JetBrainsi loodud spetsiaalne IDE Rust programmeerimiskeele jaoks. See pakub sügavat integratsiooni Rusti tööriistahelaga, sealhulgas Cargo, rustfmt ja Clippy tuge. IDE sisaldab nutikat kooditäitmist, refaktoreerimistööriistu, sisseehitatud debuggerit ning profiilimistööriistu jõudluse analüüsimiseks. Ametlik väljalase toimus 21. mail 2024.


Omadused


Koodi kiirem kirjutamine

Kontekstipõhisest koodi lõpuleviimisest ja genereerimisest,
reaalajas analüüsist ja kiirparandustest,
nutikatest refaktoritest, reaalajas mallidest ja muust.

Jooksuta, silu ja testi oma koodi

Käivita üksikuid teste,
testmooduleid või kõiki projekti teste ja
vali erinevaid käivituse eesmärke.

Halda projekte

Redigeeri TOML-faile koodianalüüsi ja
Cargo täieliku integratsiooni abil.
Samuti saad hallata kogu tööruumi ja
käivitada Cargo käske spetsiaalsest tööriistaaknast.

Roblox Studio



Mis on Roblox Studio?

Roblox studio on mängumootor, mida saab kasutada lihtsate mängude tegemiseks

See sisaldab endas:

  • Koodiredaktor
  • Objektide ehitamise/mudeleerimise aken
  • Silur (VÄGA SITT)
  • Animatsioonide redaktor
  • Digitaalne heli töötlemine


Kust see alguse sai?

Roblox Studio väljastati 23. juuli, aasta 2012 Californias
Enne seda sai mänge luua ainult läbi RobloxApp'i


Kes tegi?

Roblox Studio kaks peamist arendajat on Erik Cassel ja David Baszucki


Programmeerimiskeel

See IDE kasutab Luau Programmeerimiskeelt, mis on Lua programmeerimiskeele
eraldi "branch".


Uuendused

Tarkvara uuendatakse väga tihti, kahjuks enamus aega see toob vigu juurde.
Umbes iga poole aasta tagant nad uuendavad süsteeme koos programmeerimiskeelega,
mille pärast vanad mängud ei hakka enam tööle.

Karantiini algusest (u 2020) hakkas Roblox muutma nende IDE kujundust.
Tänapäevaks on see väga modernne ja mõnes mõttes ebamugav.
